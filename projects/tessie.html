Tessie Tesla Integration

Project Overview

Tessie provides advanced Tesla monitoring and control integrated with both Home Assistant and iOS Shortcuts. This dual-platform approach enables comprehensive vehicle automation, from intelligent climate control based on environmental conditions to location-triggered garage automation and detailed charging management.

Why Tessie Over Native Tesla Integration

After evaluating both Tesla's native app and third-party integrations, Tessie emerged as the clear winner for several key reasons:

  • Cleaner Interface: More intuitive UI compared to Tesla's native app with better organization of vehicle data
  • Richer Data Access: Significantly more data points and metrics available for monitoring and automation
  • Battery Health Monitoring: Detailed battery health tracking and degradation analysis not available natively
  • Enhanced Flexibility: More robust API for Home Assistant integration with faster polling and better reliability
  • Advanced Features: Features like historical data tracking, detailed efficiency metrics, and comprehensive automation hooks
  • Value Pricing: Legacy plan at $5.40/month (after taxes) is an absolute steal for the feature set provided

Home Assistant Automations

Intelligent Climate Control

The climate control system uses dual-threshold logic to automatically manage cabin comfort:

  • Vent Windows: Automatically opens windows when both external temperature exceeds a threshold AND internal cabin temperature exceeds a separate threshold. Prevents unnecessary window opening on hot days when parked in shade.
  • Weather Protection: Monitors local weather conditions and automatically closes windows if precipitation is detected, preventing water damage to interior
  • Location Awareness: Different temperature thresholds based on parking location (home vs. work vs. public)

Location-Based Garage Control

  • Arrival: Automatically opens garage door when vehicle arrives home using geofencing
  • Departure: Closes garage door automatically when vehicle departs to prevent leaving it open
  • Timeout Safety: If garage remains open 10 minutes after arrival, sends notification and offers to close

iOS Shortcuts Integration

Tessie's iOS Shortcuts integration provides quick-access climate control independent of Home Assistant:

  • Pre-conditioning Shortcuts: One-tap shortcuts to pre-cool or pre-heat vehicle before departure
  • Departure Routines: Combined shortcuts that unlock, set climate, and start defrost as needed
  • Widget Access: Home screen widgets for instant status checks without opening the app
  • Automation Integration: iOS automation triggers based on time, location, or calendar events

Charging Management & Monitoring

Comprehensive charging monitoring with dynamic dashboard visibility:

Dynamic Dashboard Cards

  • Conditional Display: Charging-specific cards automatically unhide when charging session starts
  • Estimated Completion Time: Live countdown to charge completion based on current charge rate
  • Power Usage Tracking: Real-time kW draw and session energy consumption
  • Efficiency Metrics: Miles added per hour, cost per kWh, total session cost

Charging Automations

  • Charge Complete Notifications: iOS push notification when battery reaches target charge level
  • Scheduled Charging: Automatically set charging schedules based on electricity rate tiers
  • Battery Health Protection: Warnings when charging above recommended daily limit (80%)

Setup Experience

Setup Difficulty: Very easy. Tessie's setup process was straightforward and well-documented.

Configuration Steps

  1. Created Tessie account and linked Tesla vehicle (2 minutes)
  2. Installed Tessie integration in Home Assistant via HACS (5 minutes)
  3. Added API key and configured polling intervals (3 minutes)
  4. Created custom dashboard cards and automations (2-3 hours of customization)

Home Assistant Integration

  • Installation Method: HACS (Home Assistant Community Store)
  • Entity Count: 50+ sensors and controls exposed to Home Assistant
  • Update Frequency: Configurable polling from 1-60 minutes (set to 5 minutes for balance of data freshness and battery drain)
  • Battery Impact: Minimal - Tesla reports approximately 1-2 miles of phantom drain per day with 5-minute polling

Pricing & Value

Current Plan: Legacy tier at $5.40/month after taxes

Value Assessment: Exceptional value considering the feature set, reliability, and depth of integration. The battery health monitoring alone justifies the cost for long-term vehicle ownership.

Cost vs. Benefit: The monthly cost is negligible compared to the convenience of automated garage control, intelligent climate management, and comprehensive vehicle monitoring. Being on a legacy plan makes this an absolute steal - current pricing is higher but still competitive.
