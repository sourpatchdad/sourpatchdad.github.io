Project Overview

Self-hosted 4K media streaming solution running on TrueNAS server. Fully automated media management pipeline using Sonarr, Radarr, Prowlarr, and Transmission, with Trakt integration for watch history tracking and automated syncing to the homepage.

Key Components

  • Plex Media Server: 4K streaming with hardware transcoding
  • Sonarr: Automated TV show management and monitoring
  • Radarr: Automated movie management and monitoring
  • Prowlarr: Unified indexer management for Sonarr and Radarr
  • Transmission: Download client with VPN protection
  • Trakt: Watch history tracking and homepage integration via GitHub Actions
